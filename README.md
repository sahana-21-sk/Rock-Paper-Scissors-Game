# Rock, Paper, Scissors (Python)

A simple, interactive command-line implementation of the classic **Rock, Paper, Scissors** game built using Python. 

## 🚀 Features

*   **Dynamic Computer Choices:** The computer randomly selects a new move every single round.
*   **Input Validation:** Keeps the game running smoothly by catching invalid inputs and prompting the user to try again.
*   **Replay Loop:** Play as many rounds as you want without having to restart the script.
*   **Clean UI:** Simple text-based separators to keep your terminal organized between rounds.

## 🛠️ How to Play

1. **Rock** beats **Scissors**
2. **Scissors** beats **Paper**
3. **Paper** beats **Rock**

### Controls
*   `r` - Rock
*   `p` - Paper
*   `s` - Scissors

## 📋 Prerequisites

To run this game, you only need to have **Python 3** installed on your system. The script relies entirely on Python's built-in `random` module, so no external dependencies are required.

## 💻 How to Run

1. **Clone or Download** this repository to your local machine.
2. Open your terminal or command prompt and navigate to the directory where the file is saved.
3. Run the script using the following command:

```bash
python rock_paper_scissors.py
